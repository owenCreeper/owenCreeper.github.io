关于Git

Git储存发生变化的文件(而不是代码行)

开发过程中的改变对应的指令:

git add/mv/rm 将工作室的改变放到staging

git commit 从staging放到本地库

git commit -a 上面两个的综合

git reset <file> staging回退到工作室

git reset <commit> 从本地库直接回退到工作室

git push 从本地库上传到网络仓库

git fetch 从网络仓库扒到本地库

git diff 比对工作室和staging的文件

git diff HEAD 比对工作室和本地库的文件

git clone/pull 直接从网络仓库拉取到工作室

分支更替的指令:

git checkout -b xxx 从当前分支分出一个并切换

git merge xxx 将当前分支与xxx合并

git branch -d xxx 删除一个分支
